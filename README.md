# 💪 LET'S BUFF UP DAWG — Anime Hypertrophy & Fitness Engine

Welcome to **LET'S BUFF UP DAWG**! This is an open-source, interactive personal training and nutrition tracking engine built entirely with Streamlit. Designed with a sleek, high-energy aesthetic, this application empowers users to calculate crucial body metrics, customize nutrition strategies, and plan out advanced training blocks seamlessly without hidden subscriptions.

---

## 🔥 Key Features

*   **📊 Dynamic Body Composition Engine:** Instantly calculates Body Mass Index (BMI), assigns categories, and tracks target weight projections using standard biometrics.
*   **🍎 Energy & Macronutrient Breakdown:** Calculates Total Daily Energy Expenditure (TDEE) using multi-factor activity scales and tracks customized deficit plans.
*   **⏳ Intermittent Fasting Scheduler:** Integrated timers and routines including Lean Gains (16:8), the Warrior Diet (20:4), and Advanced Windows.
*   **⚔️ Monthly Periodization Workout Planner:** Contains 4 comprehensive splits (including an elite **Aesthetic Bodybuilding** track) with automated monthly rotation switches between barbell strength blocks and dumbbell deep-stretch isolation blocks to prevent plateauing.
*   **🎨 Cyberpunk/Anime UI:** Fully styled using optimized custom CSS configurations for zero layout shifting and clean metrics monitoring.

---

## 🛠️ Tech Stack & Architecture

*   **Language:** Python 3.10+
*   **Framework:** Streamlit (UI & Reactive Engine)
*   **Design Paradigm:** Modular Functions & Dynamic Injected CSS
